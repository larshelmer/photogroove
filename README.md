# photogroove
"Elm in action" app

## Building

```
➜ elm make src/Main.elm --output app.js
```

## Testing

```
➜ elm-test
```

## Running

### Elm reactor
```
➜ elm reactor
```
Browse to `http://localhost:8000/index.html`

### Static webserver

```
➜ npm install -g http-server-spa
➜ http-server-spa .
```
Browse to `http://localhost:8080`
